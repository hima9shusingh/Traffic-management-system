#  Smart Traffic Management System

# Overview

A dynamic traffic control system that simulates real-time signal management based on traffic density. The system intelligently prioritizes roads and adjusts signal timing to ensure smooth and efficient traffic flow.


# Features

* Real-time traffic simulation
* Dynamic signal switching (Red → Yellow → Green)
* Priority-based road selection
* Interactive dashboard UI
* Visual traffic indicators


# How It Works

The system analyzes traffic levels across multiple roads and activates the signal for the road with the highest priority, reducing congestion and improving flow efficiency.
